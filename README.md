# ECCV 2026 Workshop Webpage

Single-page static website for:

**Event-Based Multimodal Vision: Imaging, Perception, and Understanding**

## Structure

- `index.html`: main page
- `css/style.css`: page styles
- `assets/images/`: hero visual
- `assets/avatars/`: placeholder portraits
- `Vibe/`: organizer and speaker portraits used by the page

## Usage

This is a static website and does not require build tools.

1. Open `index.html` directly in a browser, or
2. Host this folder with any static file server / GitHub Pages

## Content Updates

- Edit text and section content in `index.html`
- Replace photos in `Vibe/` or update image paths in `index.html`
- Tune visual style in `css/style.css`

## Deployment (GitHub Pages)

Upload the repository as-is and configure GitHub Pages to serve from the repository root (or the branch/folder you use for static hosting).
